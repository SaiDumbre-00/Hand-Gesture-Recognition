HAND GESTURE RECOGNITION 🖐️🎮

Real-time hand gesture detection and control using Python, OpenCV, and MediaPipe.

─────────────────────────────
FEATURES:
- Detect hand landmarks in real-time from your webcam
- Recognize gestures: thumbs up, fist, open palm
- Map gestures to actions (play/pause, volume control)
- Works with Python 3.10+

─────────────────────────────
LIBRARIES REQUIRED:
- OpenCV      : Video capture & image processing
- MediaPipe   : Fast hand landmark detection
- NumPy       : Data & array handling

─────────────────────────────
FUTURE IMPROVEMENTS:
- Add more gestures (peace sign, rock sign, etc.)
- Integrate with apps for direct control
- Optimize for faster detection on low-end devices
